# Shrey's Portfolio

A modern, responsive personal portfolio website showcasing my projects and skills.

## Project Description
This portfolio is designed to demonstrate my web development capabilities, featuring a clean modern UI, responsive layout across all devices, and an interactive project gallery. It includes sections for:
- **Home**: Hero section with introduction
- **About**: Professional summary and skills
- **Projects**: Showcase of detailed project cards and visual gallery
- **Contact**: Functional-style contact form

## Tech Stack
- **HTML5**: Semantic structure
- **CSS3**: Modern styling with Flexbox, Grid, and Custom Properties (Variables)
- **JavaScript**: Interactive elements (Mobile Menu, Animations, Theme Toggle)
- **Git**: Version control
